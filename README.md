# art-in-transit

## Deployment

This app deploys to GitHub Pages at `https://yizhe-ang.github.io/art-in-transit/`.
In the GitHub repository settings, set Pages > Build and deployment > Source to
GitHub Actions. Pushes to `main` will build and deploy `dist/`.
