# art-in-transit

This is a personal and unofficial project showcasing Singapore's **Land
Transport Authority's** (LTA) **Art in Transit** (AIT) programme in a more
exploratory, interactive and visual manner.

Visit the [Official LTA Art in Transit][lta-art-in-transit] page for more
information.

The images and information about the artworks are scraped from the
[Bloomberg Connects][bloomberg-connects] page about the programme.

[Art Outreach][art-outreach] also has a set of virtual tours where you can
learn more about the stories and artists behind these artworks.

Made by [Yi Zhe Ang][yi-zhe-ang].
